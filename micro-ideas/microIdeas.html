Micro Project Ideas

Never-Ending Popups

The system will overwhelm the user with constant pop ups that will increase each time the user exits out of one. Pop ups will eventually fill up the screen. The input is clicking to close a popup and what changes over time is the number of pop ups increase over time. Friction appears when more appear after closing one. The intended feeling is to make the user feel overwhelmed and overstimulated.

Unwinnable

The system changes the position of ur goal when you are close to succeeding. The user will control a small circle and the goal will be the triangle on the screen. Almost like Sysyphus and the boulder where it never ends and you never get the satisfaction of finishing the task. The input is moving the arrow keys to move the circle but what changes over time is the triangle will move whenever the user gets close. The friction is the close proximity to the triangle from the circle will trigger it to move to a different spot on the screen. This is supposed to feel impossible and unfair.

Unheard Voices

This system shows what it's like to have your voice unheard or dominated by others. The user presses a key to send a message which triggers other messages to appear on the screen that are bigger or bolder or cover the user's message. The input is pressing a key to send a message and what changes over time is that other messages appear and the friction is that the user's message is obscured and fades faster than the larger bold messages. The intended feeling for this is frustration and helplessness of being ignored.
